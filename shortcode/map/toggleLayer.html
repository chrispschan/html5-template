Map.toggleLayer (options, layer = -1, index = -1)

Show/hide/remove Map layer(s).

Parameters
options Type: object
Action.
layer Type: number
Edit layer index.
index Type: number
The Map index in Map.map Array. If -1, set all Maps in Map.map Array.
Map.toggleLayer.options
show Type: boolean
Hide layer(s) when set false.
remove Type: boolean
Remove layer(s) when set true.
