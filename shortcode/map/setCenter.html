Map.setCenter (center, panTo = true, index = -1)

Set the Map center.

Parameters
center Type: google.maps.LatLngLiteral
New center of the Map. Required.
panTo Type: boolean
If true, with animation for pan to new center.
index Type: number
The Map index in Map.map Array. If -1, set all Maps in Map.map Array.
